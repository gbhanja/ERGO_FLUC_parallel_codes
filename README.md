
This repository computes ergotropy, ergotropy fluctuations, and higher moments in quantum battery models:

- Tavis–Cummings (TC)
- Dicke
- Central Spin (CS)

Both standard and Kac-scaled interactions are included.

### This repository contains the following branches:

- [Dicke Operator Method](https://github.com/gbhanja/ERGO_FLUC_parallel_codes/tree/Dicke-operator-method)
- [Dicke Operator Method with Kac-scaling](https://github.com/gbhanja/ERGO_FLUC_parallel_codes/tree/Dicke-Kac-operator-method)
